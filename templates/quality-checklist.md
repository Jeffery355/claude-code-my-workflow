# Pre-Submission Quality Checklist

**Use this checklist before submitting or committing a paper to version control.**

---

## 1. Code Reproducibility

- [ ] `00_master.do` runs from start to finish without errors
- [ ] All do-files are numbered and run in correct order
- [ ] Log files are clean (no errors, warnings reviewed)
- [ ] No hardcoded paths — all paths come from globals set in `00_master.do`
- [ ] Raw data is never modified (all cleaning creates new files in `outputs/`)
- [ ] `do-files/shared/` utilities are on adopath

---

## 2. Table Reproducibility

- [ ] All numbers in LaTeX tables match the Stata output
- [ ] Tolerance check: every cell in every table passes audit-reproducibility (within tolerance)
- [ ] Table notes correctly describe: sample, clustering, significance stars
- [ ] Dependent variable mean or SD is reported in table notes
- [ ] All tables are referenced in the manuscript text ("Table 2 shows...")
- [ ] Table files are named descriptively: `T1_summary_stats.tex`, `T2_main_results.tex`

---

## 3. Figure Reproducibility

- [ ] All figures are generated by Stata or R code (no manual editing)
- [ ] Figure file format is PDF or PNG as appropriate
- [ ] Figures are labeled: `F1_main_results.pdf`, `F2_placebo.pdf`
- [ ] All figures are referenced in the manuscript text
- [ ] Axes are labeled with units
- [ ] Legends are clear and complete

---

## 4. Manuscript Completeness

- [ ] Abstract: 4 sentences (question, method, result, contribution)
- [ ] Introduction: hook, gap, contribution, roadmap
- [ ] Literature review: organized by mechanism (not by paper)
- [ ] All citations are in Bibliography_base.bib
- [ ] No `TODO` or `FIXME` placeholders in text
- [ ] Acknowledgments included (AER requires data availability statement)
- [ ] JEL codes included

---

## 5. Cross-Artifact Consistency

**Run `/verify-claims [manuscript.tex]` and `/audit-reproducibility [manuscript] [outputs/]`**

- [ ] Every numeric claim in the text has corresponding Stata output
- [ ] Every table cell in the LaTeX matches the Stata regression output
- [ ] Every figure has corresponding code that produces it
- [ ] No `\input{}` or `\ref{}` in the manuscript points to missing files

---

## 6. Identification Validity

### For DiD:
- [ ] Event study plot included
- [ ] Joint test of pre-treatment coefficients: p-value reported
- [ ] Parallel trends assumption justified in text
- [ ] Callaway-Sant'Anna estimator compared (if treatment timing varies)

### For IV:
- [ ] First stage F-stat > 10 (Stock-Yogo threshold)
- [ ] Reduced form results shown
- [ ] Overidentification test (Hansen J) reported

### For RDD:
- [ ] McCrary sorting test reported
- [ ] Placebo cutoffs tested
- [ ] Bandwidth sensitivity shown

### For Panel FE:
- [ ] Hausman test reported (FE vs RE)
- [ ] Alternative clustering levels shown

---

## 7. Robustness and Mechanism

- [ ] At least 5 robustness checks in main paper
- [ ] All robustness results reported (including ones that fail)
- [ ] Mechanism section has direct measurement AND interaction AND falsification
- [ ] Heterogeneity by mechanism tested

---

## 8. Pre-Registration

- [ ] If pre-registered: all pre-registered tests are reported
- [ ] Any non-pre-registered tests are labeled "exploratory"
- [ ] Benjamini-Hochberg correction applied if many tests

---

## 9. Final Checks

- [ ] Spelling and grammar reviewed (use `/proofread [file]`)
- [ ] Word count is appropriate for target journal
- [ ] Page limits respected (AER has no strict limit but 250-word abstract; QJE ~15,000 words)
- [ ] All authors have agreed on the final version

---

## 10. Pre-Commit Run

Before running `/commit`, run this sequence:

```bash
# 1. Run full analysis
stata -b do papers/[slug]/do-files/00_master.do

# 2. Verify table numbers match
/audit-reproducibility papers/[slug]/manuscript/[slug].tex papers/[slug]/outputs/

# 3. Verify all claims
/verify-claims papers/[slug]/manuscript/[slug].tex

# 4. Proofread
/proofread papers/[slug]/manuscript/[slug].tex

# 5. Check all checklist items above
```
